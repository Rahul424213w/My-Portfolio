# My Portfolio — RAK.

A responsive and interactive personal portfolio website built using HTML, CSS, and JavaScript.

**Live Site:** [https://rahul-ak.web.app](https://rahul-ak.web.app)

## Features

- Animated intro sequence with GSAP
- Custom cursor with magnetic text effects
- Crowd canvas walking animation
- Cinematic parallax scrolling sections
- Interactive 3D robot assistant (Spline)
- Orbiting skills visualization
- Auto-scrolling project showcase
- Dark/Light theme toggle
- Global audio control with mute toggle
- Contact form via Formspree
- Firebase Analytics integration

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Animations:** GSAP + ScrollTrigger, Three.js
- **3D:** Spline Viewer
- **Hosting:** Firebase Hosting
- **CI/CD:** GitHub Actions
- **Contact:** Formspree

## Deployment

This site is hosted on Firebase Hosting and auto-deploys on every push to `main` via GitHub Actions.
